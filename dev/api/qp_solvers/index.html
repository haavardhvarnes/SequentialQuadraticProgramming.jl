QP Subsolvers

The SQP algorithm solves a quadratic programming (QP) subproblem at each iteration. Three QP solvers are available:

SolverPackageTypeBest for
COSMOQPSolver()COSMO.jl (built-in)Interior pointDefault, small-medium QPs
ClarabelQPSolver()Clarabel.jl (extension)Interior point (conic)Alternative solver
HiGHSQPSolver()HiGHS.jl (extension)Simplex / IPMLarge sparse QPs

Abstract Type

SequentialQuadraticProgramming.AbstractQPSolver — Type
AbstractQPSolver

Abstract type for QP subproblem solvers.

COSMO (Default)

SequentialQuadraticProgramming.COSMOQPSolver — Type
COSMOQPSolver <: AbstractQPSolver

QP subproblem solver using COSMO (conic solver).

QP Methods

SequentialQuadraticProgramming.solve_qp — Function
solve_qp(solver, H, g, h, df, dg, dh, x)

Solve the QP subproblem without the delta slack variable.

min  0.5 * d' H d + df(x)' d
s.t. dg(x) d <= -g(x)     (inequality)
     dh(x) d  = -h(x)     (equality)

Returns (dx, multipliers, delta, A).

SequentialQuadraticProgramming.solve_qp_with_slack — Function
solve_qp_with_slack(solver, H, g, h, df, dg, dh, x, multiplier, rho)

Solve the QP subproblem with Schittkowski delta slack variable for infeasibility.

min  0.5 * [d; delta]' Hqp [d; delta] + [df(x); 0]' [d; delta]
s.t. dg(x) d + g(x) * delta <= -g(x)   (inequality, relaxed)
     dh(x) d                 = -h(x)    (equality)
     0 <= delta <= 1 - eps

Returns (dx, multipliers, delta, A).

SequentialQuadraticProgramming.solve_qp_trust_region — Function
solve_qp_trust_region(solver, H, g, h, df, dg, dh, x, delta_radius)

Solve the QP subproblem with trust region constraint ||d||∞ <= Δ.

min  0.5 d'Hd + df(x)'d
s.t. dg(x) d <= -g(x)
     dh(x) d  = -h(x)
     -Δ <= d_i <= Δ      (trust region)

Returns (dx, multipliers, delta_slack, A).
