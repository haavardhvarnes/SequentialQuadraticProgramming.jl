Problem Definition

NLPProblem

SequentialQuadraticProgramming.NLPProblem — Type
NLPProblem{T, F, G, H, V}

Nonlinear programming problem: min f(x) s.t. g(x) <= 0, h(x) = 0.

Variable bounds are converted to inequality constraints internally.
