Hessian Updates

The solver maintains an approximation $H$ of the Hessian of the Lagrangian. The update strategy is:

  1. Try analytical Hessian (ForwardDiff or external) - use if positive definite
  2. L-BFGS reconstruction from stored (s, y) history pairs
  3. Robust BFGS update (Yang et al. 2022)
  4. Fall back to identity matrix

Robust BFGS

SequentialQuadraticProgramming.robust_bfgs_update! — Function
robust_bfgs_update!(H, s, y; m=1e-5, M=1e5)

Robust BFGS update from Yang et al. 2022.

Modifies and returns H in-place.

Reference: "A robust BFGS algorithm for unconstrained nonlinear optimization problems" https://arxiv.org/pdf/1212.5929.pdf

L-BFGS

SequentialQuadraticProgramming.lbfgs_hessian — Function
lbfgs_hessian(s_history, y_history, n)

Reconstruct an approximate Hessian from L-BFGS history using the compact representation. Returns a dense positive definite matrix.

Uses the two-loop recursion approach to build H₀ scaled by γ = (sᵀy)/(yᵀy) from the most recent pair, then applies all stored corrections.

Utilities

SequentialQuadraticProgramming.ensure_positive_definite! — Function
ensure_positive_definite!(H)

If H is not positive definite, replace it with the identity matrix. Returns (H, was_modified).

SequentialQuadraticProgramming.update_hessian! — Function
update_hessian!(H, s, y, iteration, k_reset)

Perform BFGS Hessian update using robust BFGS, with initial scaling on first iteration. Returns (H, k_reset).
